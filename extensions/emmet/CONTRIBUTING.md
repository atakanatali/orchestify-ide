## How to build and run from source?

Read the basics about extension authoring from [Extending Orchestify](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run Orchestify from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Orchestify
- Open the `extensions/emmet` folder in the vscode repo in Orchestify
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of Orchestify.

- Read [Build and Run Orchestify from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for Orchestify
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
